# Portfolio (React + Vite)

This portfolio was migrated from static HTML/CSS/JS to a React.js project using Vite.

## Scripts

- `npm run dev`: start local development server
- `npm run build`: create production build in `dist/`
- `npm run preview`: preview production build locally

## Structure

- `src/App.jsx`: main React portfolio page
- `src/App.css`: styles migrated from the original project
- `public/img/`: image assets
- `public/pdf/`: CV and certificate PDFs

## Notes

The original static behavior was migrated into React effects (theme toggle, animated tooltips, cursor glow, progress ring, section reveal, and typed about text).
